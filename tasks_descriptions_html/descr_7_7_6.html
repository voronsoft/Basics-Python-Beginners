Подвиг 6. В программе объявлен следующий многомерный список:

d = [1, 2, [True, False], ["Сидней", "Мельбурн", [100, 101], ['True', [-2, -1]]], 7.89]

С помощью рекурсивной функции get_line_list необходимо на его основе создать одномерный список из значений элементов списка d. Функция должна возвращать новый созданный одномерный список.

Только объявить со следующей сигнатурой:

def get_line_list(d,a=[]): ...

где d - исходный список; a - новый формируемый.

Вызывать функцию не нужно. Выводить на экран ничего не нужно.

Подвиг 1. Объявите внешнюю функцию с именем counter_add и следующей сигнатурой:

def counter_add(): ...

В теле функции counter_add создайте локальную переменную add_value=5. Затем объявите вложенную функцию с одним параметром. Эта вложенная функция должна увеличивать значение переданного аргумента на значение переменной add_value и возвращать результат.

Внешняя функция counter_add должна возвращать ссылку на вложенную функцию. Таким образом, будет создано замыкание — вложенная функция должна использовать переменную add_value из внешней области.

После этого вызовите функцию counter_add и результат её вызова присвойте переменной cnt. Затем вызовите вложенную функцию через переменную cnt, передав ей значение k, считанное из входного потока: 

k = int(input())

Выведите результат вызова вложенной функции на экран.

Sample Input: 

7

Sample Output: 

12

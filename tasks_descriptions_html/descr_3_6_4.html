Подвиг 4. На вход программе подается строка с названиями городов, записанных в одну строчку через пробел. Необходимо прочитать эту строку и на ее основе сформировать список с помощью следующей команды: 

cities = input().split()

Затем, необходимо проверить, присутствует ли в этом списке город "Мадрид". Вывести на экран True, если присутствует и False - в противном случае. Решить эту задачу следует без использования условного оператора.

Sample Input : 

Стокгольм Осло Копенгаген Мадрид

Sample Output: 

True

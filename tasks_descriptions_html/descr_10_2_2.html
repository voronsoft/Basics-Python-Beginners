Как включить третий бит (простое объяснение)

Любое число в компьютере хранится в виде нулей и единиц — это называются биты.

Например, число 22 в двоичном (битовом) виде выглядит вот так:

бит №:      4 3 2 1 0
число:       1 0 1 1 0

Биты нумеруются справа налево, начиная с нуля.

Задача:

Включить третий бит — это значит, что мы должны сделать его равным 1, даже если он был равен 0.

Как это сделать?

Мы используем битовую операцию ИЛИ, которая записывается как | в Python.

Вот простая формула:

n | (1 << 3)

Что здесь происходит?

  • 1 << 3 — это число, в котором только третий бит равен 1 (то есть 8 в десятичной системе).
  • | — "ИЛИ": если хотя бы один бит равен 1 — результат 1.
Пример:

Пусть n = 2. Это число в битах:

бит №:      4 3 2 1 0
число:       0 0 0 1 0

Третий бит (с номером 3) сейчас равен 0. Включим его:

2 | (1 << 3) = 2 | 8 = 10

Теперь число выглядит так:

бит №:      4 3 2 1 0
новое:      0 1 0 1 0
Готовый код на Python:
n = int(input())
print(n | (1 << 3))

Он работает так: читает число, включает третий бит и выводит результат.

Полезно знать: Эта техника часто используется, когда нужно "включить лампочку" внутри числа — например, активировать флаг, признак или команду.

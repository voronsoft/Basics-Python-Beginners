Подвиг 3. На вход программе подаются три целых числа, записанных в одну строку через пробел. Необходимо прочитать эти числа и сформировать список lst, хранящий прочитанные значения в порядке их чтения.

Результат (список) выведите на экран, используя команду:

print(lst)

Sample Input: 

8 11 3

Sample Output: 

[8, 11, 3]

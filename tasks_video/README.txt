Если видео файл для занятия в папке tasks_video не найден,
приложение переключится на воспроизведение видео с YouTube открыв браузер ОС Windows назначенного по умолчанию.

!!! ФАЖНО для работы приложения в автономном режиме.
Файлы с видео для занятий можно скачать с google-drive по адресу:
https://drive.google.com/file/d/17YC7R-i_D0Ntimtfeuo0Jj3A9VKnNvNF/view?usp=drive_link
Распакуйте видео в папку tasks_video по пути: C:\Program Files (x86)\Basics Python Beginners\tasks_video
все будет работать автономно во встроенном плеере ОС Windows.
